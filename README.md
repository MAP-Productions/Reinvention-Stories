# Reinvention

### Install at the web root or sub-domain, not in a sub-directory.



## Setup


```bash
npm install
```

```bash
git submodule update --init --recursive
```

## Build Commands

```bash
bbb watch
```

```bash
bbb debug
```

```bash
bbb release
```

## Localize Video Resources

```bash
cd app/video && sudo chmod +x get.sh && ./get.sh
```
